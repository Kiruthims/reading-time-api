# Reading Time API

A professional, tiered REST API to calculate reading time for any text content. Implements Medium.com's user-friendly logic: always rounds up to the nearest minute.

## Features

- **Accurate Calculation**: Uses regex-based word counting for precision.
- **Tiered for Every Need**: From hobbyist blogs to enterprise content pipelines.
- **Clean JSON API**: Simple integration with any stack.
- **Bulk Processing**: Analyze multiple texts in a single request (Pro+ tiers).
- **Customizable Speed**: Adjust words-per-minute for different audiences.
